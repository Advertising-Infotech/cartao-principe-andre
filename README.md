# Cartão Príncipe André Luís

Digital business card built with Next.js, featuring multilingual support and modern UI.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Internationalization:** i18next
- **Icons:** Lucide React
- **Hosting:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Features

- 7 languages supported (Portuguese, English, Hebrew, Arabic, Russian, Chinese, Spanish)
- RTL support for Hebrew and Arabic
- Digital business card with vCard download
- Responsive design with glassmorphism effects
- Interactive carousel with honors and certifications
