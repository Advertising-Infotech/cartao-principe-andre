# Sessão OpenCode - Cartão Príncipe André Luís
**Data:** 21 de Março de 2026
**Status:** Projeto migrado de Vite/React para Next.js 15

---

## Decisões Técnicas Tomadas

### Stack Escolhida
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **i18n:** i18next + react-i18next
- **Icons:** Lucide React
- **Hosting:** Vercel

### Estrutura de Diretórios Criada
```
src/
├── app/
│   ├── globals.css      # Estilos globais + animações
│   ├── layout.tsx      # Layout principal + Providers
│   └── page.tsx        # Página principal
├── components/
│   ├── ActionGrid.tsx   # Botões de contato (WhatsApp, LinkedIn, etc)
│   ├── FeaturedProperty.tsx  # Carrossel de honrarias
│   ├── Footer.tsx       # Rodapé
│   ├── ProfileHeader.tsx # Header com foto, idiomas, redes sociais
│   └── Providers.tsx    # Wrapper i18n
└── i18n/
    └── index.ts         # Traduções em 7 idiomas

public/
└── carrossel/           # 52 imagens + 7 arquivos de tradução JSON
```

### Layout Mantido Exatamente Igual
- Cores: `#D4AF37` (dourado), fundo `#000` (preto)
- Glassmorphism: `bg-white/10 backdrop-blur-xl`
- Animações CSS: `blink-name` e `blink-yellow`
- Fonte: system sans com peso bold tracking-tight
- Bordas: `border-4 border-[#D4AF37]` na foto do perfil

### Idiomas Suportados
1. Português (pt) - default
2. Inglês (en)
3. Hebraico (he) - RTL
4. Árabe (ar) - RTL
5. Russo (ru)
6. Chinês (zh)
7. Espanhol (es)
